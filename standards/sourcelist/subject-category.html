Introduction

Subject Category Sources identifies subject category lists and assigns a code to each. The purpose of these source codes is to specify the source of subject category codes contained in metadata records. For code assignment, general structure, usage, and maintenance guidelines see Source Codes for Vocabularies, Rules, and Schemes.

Some subject category lists, such as the Library and Archives Canada collections and the AGRICOLA subject categories, also contain schemes that  that can also be considered classification schemes.  Thus some of the subject category sources assigned source codes in this list may also be used in usage elements identified for classification schemes in Classification Scheme Source Codes, if appropriate.

Arrangement of the List

Entries in the list are arranged in alphabetical order by the code and consist of the source code followed by the bibliographic citation for the source documentation. The citations are linked to the online source for some codes.

Code Structure

Each code consists of a maximum of 12 lowercase alphabetic characters. Possible additions to codes are described in Source Codes.
