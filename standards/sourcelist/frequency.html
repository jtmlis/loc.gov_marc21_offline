Introduction

Frequency of Issue Sources contains a list of standard code and/or term sources for specifying the frequency of issue for a resource and assigns a code to each source. The purpose of this list is to enable the frequency of issue vocabulary in metadata records to be identified by a code. For code assignment, general structure, usage, and maintenance guidelines see Source Codes for Vocabularies, Rules, and Schemes.

Arrangement of the List

Entries in the list are arranged in alphabetical order by the code and consist of the source code followed by the bibliographic citation for the source documentation. The citations are linked to the online source for some codes.

Code Structure

Each code consists of a maximum of 14 lowercase alphabetic characters. Possible additions to codes are described in Source Codes.
