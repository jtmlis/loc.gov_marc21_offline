IN Harmony Sheet Music Cataloging Tool (Indiana University)

Description: This open source cataloging tool is designed specifically for sheet music.It is available for download as Windows and Mac installers, and as source code. The tool can be set up to operate on a single desktop or use a shared
database. It has the capability to export records in MODS and simple Dublin
Core.

Tool URL: http://inharmonycat.sourceforge.net/

Last update: August 06, 2009


SobekCM

Description: SobekCM allows users to discover online resources via semantic and full-text searches, as well as a variety of different browse mechanisms. For each digital resource in the repository there are a plethora of display options, which may be selected by an appropriately authenticated use. This repository includes online metadata editing and online submissions in support of institutional repositories. In addition to the repository system, with full online tools, there are related tools are windows applications which can run either stand-alone or in support of a SobekCM instance which also support MODs. For MODs, the SobekCM METS Editor is especially popular: http://ufdc.ufl.edu/software/mets . See all SobekCM related tools: http://ufdc.ufl.edu/software/tools

Tool URL: http://sobekrepository.org/

Documentation URL: http://sobekrepository.org/

Last update: January 08, 2014
