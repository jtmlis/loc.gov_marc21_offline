Introduction

The following Value Lists for codes and controlled vocabularies are maintained online by the Library of Congress.  These Value Lists may be used wherever there is a need to use a controlled vocabulary.

Many data elements in MARC and MODS/MADS contain values that may come from these controlled lists.  Each list indicates where editorial content control of the list resides. 

See also the Source Codes for Vocabularies, Rules, and Schemes for codes that identify these and other code and term value lists that can be used in MARC and MODS/MADS records.
