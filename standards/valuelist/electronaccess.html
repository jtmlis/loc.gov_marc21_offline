Electronic Access Methods Code List

List identifier: electronaccess

Code List

This list was originally developed for use in MARC 21 Format for Bibliographic Data, MARC 21 Format for Authority Data, MARC 21 Format for Holdings Data, MARC 21 Format for Classification Data, MARC 21 Format for Community Information Data, field 856 subfield 2 (Electronic Location and Access / Access method).
