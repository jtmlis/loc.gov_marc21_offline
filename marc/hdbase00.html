MARC: News and Announcements (June 15, 2000)

TECHNICAL NOTICE (June 15, 2000)

Format for Holdings Data

Format Changes List
Base Text (January 2000)

Network Development and MARC Standards Office
Library of Congress

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Holdings Data. It can also be found in Appendix F of the MARC 21 Format for Holdings Data.

This edition of the MARC 21 Format for Holdings Data supersedes the December 1989 base text and updates No. 1 (April 1991), No. 2 (July 1994), and No. 3 (October 1998) to the USMARC Format for Holdings Data as well as the November 1990 base text and Amendment No. 1 (January 1992) to the Canadian MARC Communication Format: Holdings Data. It includes changes resulting from proposals which were considered by the ALA ALCTS/LITA/RASD Machine-Readable Bibliographic Information Committee (MARBI) at its meetings from January 1999 through January 2000 and by the Canadian Committee on MARC (CCM) at its meetings from 1998 through January 2000.

LC Implementation Plans
Subscribers can anticipate receiving MARC records reflecting these changes in all distribution services not earlier than September 13, 2000.

  • Deleted content designators:
    • Field
      • 023 Standard Film Number
    • Subfield code
      • 856 $g (Uniform Resource Name) [Incorporated in 856 $u]
  • New content designators:
    • Fields
      • 007 Physical Description Fixed Field (Tactile Material)
      • 040 Record Source
    • Subfield code
      • $2 Source of term in field 583 (Action Note)
    • Code values
      • Leader ­ character position 06 (Type of record): code u (Unknown)
      • Leader ­ character position 09 (Character coding scheme): codes # (MARC-8), a (UCS/Unicode)
      • Leader ­ character position 17 (Encoding level): code u (Unknown)
      • 007 Computer file ­ character positions 06­08 (Image bit depth), 09 (File formats), 10 (Quality assurance targets), 11 (Antecedent/source), 12 (Level of compression), 13 (Reformatting quality)
  • Changes in repeatability:
    • Subfield codes
      • $k Call number prefix in 852 (Location): changed from Not repeatable (NR) to Repeatable (R)
      • $m Call number prefix in 852 (Location): changed from Not repeatable (NR) to Repeatable (R)
  • Change in content designator names:
    • Subfield code
      • 856 $u Uniform Resource Locator Identifier
