Appendix F - Format Changes

MARC 21 Authority
April 2014

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Authority Data.

MARC 21 FORMAT FOR AUTHORITY DATA

FORMAT CHANGE LIST

Update No. 18, April 2014
This page documents the changes contained in the eighteenth update to the 1999 edition of the MARC 21 Format for Authority Data. Update No. 18 (April 2014) includes changes resulting from proposals that were considered by the MARC 21 community during January 2014.
UPDATE 18 CHANGES
■ Changes in content designator names and definitions:
Subfields
$cLocation of meeting changed to repeatable in:
         110 (Heading-Corporate Name),
         111 (Heading-Meeting Name),
         410 (See From Tracing-Corporate Name),
         411 (See From Tracing-Meeting Name),
         510 (See Also From Tracing-Corporate Name),
         511 (See Also From Tracing-Meeting Name),
         710 (Established Heading Linking Entry-Corporate Name),
         711 (Established Heading Linking Entry-Meeting Name).

UPDATE 18 LOCAL PRINT GUIDE

The following sections and fields were created/changed in Update No. 18 and should be printed to keep a print copy of the format up-to-date. Changes appear in red.

  • X10 - Corporate Names-General Information

  • X11 - Meeting Names-General Information

  • 110 - Heading-Corporate Name

  • 111 - Heading-Meeting Name

  • 410 - See From Tracing-Corporate Name

  • 411 - See From Tracing-Meeting Name

  • 510 - See Also From Tracing-Corporate Name

  • 511 - See Also From Tracing-Meeting Name

  • 710 - Established Heading Linking Entry-Coporate Name

  • 711 - Established Heading Linking Entry-Meeting Name
