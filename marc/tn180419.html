TECHNICAL NOTICE (April 19, 2018)

Additions to the MARC Country and Geographic Area Code Lists

Network Development and MARC Standards Office

As a result of a request from the British Library, new country and geographic area codes were created for the following crown dependencies: Guernsey, Isle of Man, and Jersey. Previously, these crown dependencies had been coded as part of the United Kingdom, using the assigned country code uik (United Kingdom Misc. Islands) and GAC e-uk-ui (Great Britain Miscellaneous Island Dependencies). As these three island territories are self-governing crown dependencies that do not form part of the United Kingdom, they have each been assigned their own country and geographic area code.

The Orkney and Shetland Islands, which were previously coded country code uik (United Kingdom Misc. Islands), are first order administrative divisions of Scotland, and have been reassigned to country code stk (Scotland).

The following codes have been made obsolete for new records: country code uik (United Kingdom Misc. Islands) and GAC e-uk-ui (Great Britain Miscellaneous Island Dependencies). These codes will remain valid only for items prior to April 2018.

 

1. MARC country code changes

The new country codes are:

  • gg - Guernsey
  • im - Isle of Man
  • je - Jersey

These three crown dependencies were previously coded uik for United Kingdom Misc. Islands before April 2018.

The Orkney and Shetland Islands, which were previously coded uik (United Kingdom Misc. Islands) have been reassigned to country code stk (Scotland).

 

2. MARC geographic area code changes

The new geographic area codes are:

  • e-gg - Guernsey
  • e-im - Isle of Man
  • e-je - Jersey

These three crown dependencies were previously coded e-uk-ui for Great Britain Miscellaneous Island Dependencies before April 2018.

The Orkney and Shetland Islands retain their currently assigned geographic area code: e-uk-st for Scotland.

Subscribers can anticipate receiving MARC records reflecting these changes in all distribution services not earlier than June 19, 2018. Questions regarding the country and geographic area codes should be directed to:
