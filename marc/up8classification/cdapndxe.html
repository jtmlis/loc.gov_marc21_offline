Appendix E - Format Changes

MARC 21 Classification
October 2007
This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Classification Data.

MARC 21 FORMAT FOR CLASSIFICATION DATA

FORMAT CHANGE LIST

Update No. 8, October 2007
This page documents the changes contained in the eighth update to the 2000 edition of the MARC 21 Format for Classification Data. Update No. 8 (October 2007) includes changes resulting from proposals that were considered by the MARC 21 community during 2007.
■ New content designators:
Indicator
7532nd indicator (Type of term or name):
# - No information provided
0 - Topical term
1 - Personal name
2 - Corporate name
3 - Meeting name
4 - Chronological term
5 - Geographic name
6 - Genre/form term
Subfield code
$0Authority record control number in 754 (Index term-Faceted topical terms)
