Appendix F - Format Changes

MARC 21 Authority
April 2016

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Authority Data.

MARC 21 FORMAT FOR AUTHORITY DATA

FORMAT CHANGE LIST

Update No. 22, April 2016
This page documents the changes contained in the twenty-second update to the 1999 edition of the MARC 21 Format for Authority Data. Update No. 22 (April 2016) includes changes resulting from proposals that were considered by the MARC 21 community during January 2016.

Format changes in previous updates can be found on the MARC Format Overview page.
UPDATE 22 CHANGES
■ New content designators:
Subfields
$r Total number of individuals performing alongside ensembles (NR) in 382
         (Medium of Performance)
$t Total number of ensembles (NR) in 382 (Medium of Performance)
■ Changes in content designator names and definitions:
Subfields
$e Number of ensembles renamed Number of ensembles of the same type in 382
         (Medium of Performance)

UPDATE 22 LOCAL PRINT GUIDE

The following sections and fields were created/changed in Update No. 22 and should be printed to keep a print copy of the format up-to-date. Changes appear in red.

  • 382 - Medium of Performance
