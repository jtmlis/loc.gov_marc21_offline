TECHNICAL NOTICE (December 20, 2000)

Format for Holdings Data

Format Changes List
Update No. 1 (October 2000)

Network Development and MARC Standards Office
Library of Congress

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Holdings Data. It can also be found in Appendix F of the MARC 21 Holdings Format.

This page documents the changes contained in the first update to the 2000 edition of the MARC 21 Format for Holdings Data. Update No. 1 (October 2000) includes changes resulting from proposals which were considered by the ALA ALCTS/LITA/RASD Machine-Readable Bibliographic Information Committee (MARBI) at its meetings in July 2000 and by the Canadian Committee on MARC (CCM) at its meetings in July 2000.

LC Implementation Plans
Subscribers can anticipate receiving MARC records reflecting these changes in all distribution services not earlier than March 20, 2001.

  • New content designators
    • Subfield code
      • $y Link text in 856 (Electronic Location and Access)
      • $z Numbering scheme in 853-855 (Captions and Patterns)
  • Changes in content designator names:
    • Code
      • 008/06 - Character position 2 Received and complete or ceased (was Completed)
