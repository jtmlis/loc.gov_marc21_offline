Appendix G - Format Changes

MARC 21 Bibliographic
April 2015

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Bibliographic Data.

MARC 21 FORMAT FOR BIBLIOGRAPHIC DATA

FORMAT CHANGE LIST

Update No. 20, April 2015
This page documents the changes contained in the twentieth update to the 1999 edition of the MARC 21 Format for Bibliographic Data. Update No. 20 (April 2015) includes changes resulting from proposals that were considered by the MARC 21 community during January/February 2015.

Format changes in previous updates can be found on the MARC Format Overview page.
UPDATE 20 CHANGES
■ New content designators:
Indicators
0371st indicator (Source of acquisition sequence):
# - Not applicable/No information provided/Earliest
2 - Intervening
3 - Current/Latest
Subfields
$e Number of ensembles (R) in 382 (Medium of Performance)
$3 Materials specified (NR) in 037 (Source of Acquisition)
$5 Institution to which field applies (R) in 037 (Source of Acquisition)
Fields
884Description Conversion Information (R)

■ Changes in content designator names and definitions:
Fields
088Report Number scope broadened to include series report numbers by deletion of
          the sentence: "Not used to record a number associated with a series statement."

■ Editorial modifications:

The Content Designator History notes that "University presses" was removed as an example of a type of government publication in 008/28 (006/11) in the following fields:

Fields
008Books
008Computer files
008Maps
008Continuing Resources
008Visual Materials

The following sections and fields were created/changed in Update No. 20 and should be printed to keep a print copy of the format up-to-date. Changes appear in red.
