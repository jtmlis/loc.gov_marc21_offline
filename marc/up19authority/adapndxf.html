Appendix F - Format Changes

MARC 21 Authority
October 2014

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Authority Data.

MARC 21 FORMAT FOR AUTHORITY DATA

FORMAT CHANGE LIST

Update No. 19, October 2014
This page documents the changes contained in the nineteenth update to the 1999 edition of the MARC 21 Format for Authority Data. Update No. 19 (October 2014) includes changes resulting from proposals that were considered by the MARC 21 community during June 2014.

Format changes in previous updates can be found on the MARC Format Overview page.

NON-UPDATE RELATED EDITORIAL MODIFICATIONS

The following fields do not have technical changes but contain editorial modifications (in red):

  • Subfield $s (Total number of performers) in field 382 (Medium of Performance) was incorrectly designated as repeatable (R) when the subfield was defined in Update 14 (April 2012). A correction to the documentation has been made in line with the definition of the subfield, and $s is now non-repeatable (NR).
