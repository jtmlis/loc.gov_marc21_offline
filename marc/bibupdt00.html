TECHNICAL NOTICE (December 20, 2000)

Format for Bibliographic Data

Format Changes List
Update No. 1 (October 2000)

Network Development and MARC Standards Office
Library of Congress

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Bibliographic Data. It can also be found in Appendix G of the MARC 21 Bibliographic Format.

This page documents the changes contained in the first update to the 1999 edition of the MARC 21 Format for Bibliographic Data. Update No. 1 (October 2000) includes changes resulting from proposals which were considered by the ALA ALCTS/LITA/RASD Machine-Readable Bibliographic Information Committee (MARBI) at its meetings from 1999 through July 2000 and by the Canadian Committee on MARC (CCM) at its meetings from 1999 through July 2000.

LC Implementation Plans
Subscribers can anticipate receiving MARC records reflecting these changes in all distribution services not earlier than March 20, 2001.

  • Obsolete content designators:
    • Subfield Codes
      • $g Uniform Resource Name in 856 (Electronic Location and Access)
  • New content designators
    • Subfield codes
      • $j Attribution qualifier in X00 (Personal Names)
      • $u Uniform Resource Identifier in 505 (Formatted Contents Note)
      • $u Uniform Resource Identifier in 514 (Data Quality Note)
      • $u Uniform Resource Identifier in 520 (Summary, Etc.)
      • $u Uniform Resource Identifier in 530 (Additional Physical Form Available Note)
      • $u Uniform Resource Identifier in 545 (Biographical and Historical Data)
      • $u Uniform Resource Identifier in 552 (Entity and Attribute Information Note)
      • $u Uniform Resource Identifier in 555 (Cumulative Index/Finding Aids Note)
      • $u Uniform Resource Identifier in 583 (Action Note)
      • $y Link text in 856 (Electronic Location and Access)
      • $z Display note in 514 (Data Quality Note)
      • $z Display note in 552 (Entity and Attribute Information Note)
      • $2 Source of term in 583 (Action Note)
      • $6 Linkage in 050 (Library of Congress Call Number)
    • Code Values
      • sanb South African National Bibliography Project in
        field 042 (Authentication Code)
  • Changes in repeatability:
    • 522 Geographic Coverage Note changed from Not repeatable (NR) to Repeatable (R)

  • Changes in content designator field names:
    • $u Uniform Resource Location Identifier
