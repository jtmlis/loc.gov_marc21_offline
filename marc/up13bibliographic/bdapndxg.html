Appendix G - Format Changes

MARC 21 Bibliographic
September 2011

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Bibliographic Data.

MARC 21 FORMAT FOR BIBLIOGRAPHIC DATA

FORMAT CHANGE LIST

Update No. 13, September 2011
This page documents the changes contained in the thirteenth update to the 1999 edition of the MARC 21 Format for Bibliographic Data. Update No. 13 (September 2011) includes changes resulting from proposals that were considered by the MARC 21 community during January and June 2011.

UPDATE 13 LOCAL PRINT GUIDE

The following sections and fields were created/changed in Update No. 13 and should be printed to keep a print copy of the format up-to-date. Changes appear in red.

  • 034 - Coded Cartographic Mathematical Data

  • 041 - Language Code

  • 043 - Geographic Area Code

  • 082 - Dewey Decimal Classification Number

  • 083 - Additional Dewey Decimal Classification Number

  • 084 - Other Classification Number

  • 25X-28X - Edition, Imprint, Etc. Fields-General Information

  • 264 - Production, Publication, Distribution, Manufacture, and Copyright Notice

  • 3XX - Physical Description, Etc. Fields

  • 340 - Physical Medium

  • 344 - Sound Characteristics

  • 345 - Projection Characteristics of Moving Image

  • 346 - Video Characteristics

  • 347 - Digital File Characteristics

  • 377 - Associated Language

  • 383 - Numeric Designation of Musical Work

The following fields do not have technical changes but contain editorial modifications (in red):

  • 008 - All Materials

  • 260 - Publication, Distribution, etc. (Imprint)
