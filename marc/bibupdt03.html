MARC: News and Announcements (November 6, 2002)

TECHNICAL NOTICE (November 6, 2002)

Format for Bibliographic Data

Format Changes List
Update No. 3 (October 2002)

Network Development and MARC Standards Office
Library of Congress

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Bibliographic Data. It can also be found in Appendix G of the MARC 21 Format for Bibliographic Data.

This page documents the changes contained in the third update to the 1999 edition of the MARC 21 Format for Bibliographic Data. Update No. 3 (October 2002) includes changes resulting from proposals which were considered by the ALA ALCTS/LITA/RUSA Machine-Readable Bibliographic Information Committee (MARBI) at its meetings in 2002 and by the Canadian Committee on MARC (CCM) at its meetings in 2002.

LC Implementation Plans
Subscribers can anticipate receiving MARC records reflecting these changes in all distribution services no earlier than February 6, 2003.

  • New content designators:
    • Fields
      • 026 - Fingerprint identifier
      • 038 - Record content licensor
      • 563 - Binding information
      • 648 - Subject added entry–Chronological term
    • Indicators
      • 052 - 1st indicator (Code source): value 1 (U.S. Dept. of Defense Classification)
      • 655 - 2nd indicator (Thesaurus): values 0 (Library of Congress Subject Headings); 1 (LC subject headings for children's literature); 2 (Medical Subject Headings); 3 (National Agriculture Library subject authority file); 4 (Source not specified); 5 (Canadian Subject Headings); 6 (Répertoire de vedettes-matiére)
    • Subfield codes
      • $j - Date resource modified in 046 (Special coded dates)
      • $k - Beginning or single date created in 046 (Special coded dates)
      • $l - Ending date created in 046 (Special coded dates)
      • $m - Beginning of date valid in 046 (Special coded dates)
      • $n - End of date valid in 046 (Special coded dates)
      • $2 - Source of date in 046 (Special coded dates)
      • $u - Uniform Resource Identifier in 506 (Restrictions on access note)
      • $u - Uniform Resource Identifier in 540 (Terms governing use and reproduction note)
      • $6 - Linkage in 018 (Copyright article-fee code)
    • Code values
      • Field 006 Books - character position 07-10 (Nature of contents): code u (Standards/specifications)
      • Field 006 Music - character position 4 (Music parts): codes d (Instrumental and vocal parts); e (Instrumental parts); f (Vocal parts); n (Not applicable); u (Unknown); # (No parts in hand or not specified)
      • Field 006 Music - character position 16 (Transposition and arrangement): a (Transposition); b (Arrangement); c (Both transposed and arranged); n (Not applicable); u (Unknown); # (Not arrangement or transposition or not specified)
      • Field 006 Serials - character position 3 (ISSN center): code 2 (United Kingdom)
      • Field 006 Serials - character position 7 (Nature of entire work): code u (Standards/specifications)
      • Field 006 Serials - character position 08-10 (Nature of contents): code u (Standards/specifications)
      • Field 007 Sound recordings - character position 10 (Kind of material): code b (Cellulose nitrate); code c (Acetate tape with ferrous oxide); code g (Glass with lacquer); code i (Aluminum with lacquer); r (Paper with lacquer or ferrous oxide); z (Other)
      • Field 008 Books - character position 24-27 (Nature of contents): code u (Standards/specifications)
      • Field 008 Music - character position 21 (Music parts): codes d (Instrumental and vocal parts); e (Instrumental parts); f (Vocal parts); n (Not applicable); u (Unknown); # (No parts in hand or not specified)
      • Field 008 Music - character position 33 (Transposition and arrangement): a (Transposition); b (Arrangement); c (Both transposed and arranged); n (Not applicable); u (Unknown); # (Not arrangement or transposition or not specified)
      • Field 008 Serials - character position 20 (ISSN center): code 2 (United Kingdom)
      • Field 008 Serials - character position 24 (Nature of entire work): code u (Standards/specifications)
      • Field 008 Serials - character position 25-27 (Nature of contents): code u (Standards/specifications)

  • Obsolete content designators:
    • Indicator
      • 052 - 1st indicator (Code source): value 0 (U.S. Dept. of Defense Classification)
    • Code value
      • Field 007 Sound recordings - character position 10 (Kind of material): code n (Not applicable)
    Changes in repeatability:
    • Field
      • 046 - Special coded dates: changed from Not repeatable (NR) to Repeatable (R)
    • Subfield codes
      • $k - Call number prefix in 852 (Location): changed from Not repeatable (NR) to Repeatable (R)
      • $m - Call number suffix in 852 (Location): changed from Not repeatable (NR) to Repeatable (R)

  • Changes in content designator names:
    • Fields
      • 006 - Computer files Computer files/Electronic resources
      • 006 - Serials Continuing resources
      • 007 - Computer file Electronic resource
      • 008 - Serials Continuing resources
      • 247 - Former title or title variations
    • Subfield code
      • $a in 247 (Former Title or Title Variations): Title
    • Coded values
      • 006/00 (Computer file/Electronic resources) code m: Computer File Computer file/Electronic resource
      • 006/00 (Serials) code s: Serial Serial/Integrating resource
      • 006/05 (Books) code c: Pre-adolescent Elementary and junior high
      • 006/05 (Books) code d: Adolescent Secondary (senior high)
      • 006/05 (Computer files) code c: Pre-adolescent Elementary and junior high
      • 006/05 (Computer files) code d: Adolescent Secondary (senior high)
      • 006/05 (Music) code c: Pre-adolescent Elementary and junior high
      • 006/05 (Music) code d: Adolescent Secondary (senior high)
      • 006/05 (Visual materials) code c: Pre-adolescent Elementary and junior high
      • 006/05 (Visual materials) code d: Adolescent Secondary (senior high)
      • 007/10 (Sound recording) code a: Lacquered coating
      • 007/10 (Sound recording) code m: Metal and plastic Plastic with metal
      • 007/11 (Computer file) code c: File reproduced from an computer file electronic resource
      • 008/06 (All) code c: Continuing resource Serial item currently published
      • 008/06 (All) code d: Continuing resource Serial item ceased publication
      • 008/06 (All) code u: Continuing resource Serial item status unknown
      • 008/22 (Books) code c: Pre-adolescent Elementary and junior high
      • 008/22 (Books) code d: Adolescent Secondary (senior high)
      • 008/22 (Computer files) code c: Pre-adolescent Elementary and junior high
      • 008/22 (Computer files) code d: Adolescent Secondary (senior high)
      • 008/22 (Music) code c: Pre-adolescent Elementary and junior high
      • 008/22 (Music) code d: Adolescent Secondary (senior high)

      • 008/22 (Visual materials) code d: Adolescent Secondary (senior high)
