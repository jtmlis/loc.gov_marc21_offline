TECHNICAL NOTICE (December 20, 2000)

Format for Community Information

Format Changes List
Update No. 1 (October 2000)

Network Development and MARC Standards Office
Library of Congress

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Community Information. It can also be found in Appendix F of the MARC 21 Format for Community Information.

This page documents the changes contained in the first update to the 2000 edition of the MARC 21 Format for Community Information. Update No. 1 (October 2000) includes changes resulting from proposals which were considered by the ALA ALCTS/LITA/RASD Machine-Readable Bibliographic Information Committee (MARBI) at its meetings in July 2000 and by the Canadian Committee on MARC (CCM) at its meetings in July 2000.

LC Implementation Plans
Subscribers can anticipate receiving MARC records reflecting these changes in all distribution services not earlier than March 20, 2001.

  • Obsolete content designators:
    • Field
      • 058 Other Geographic Classification Code
  • New content designators
    • Indicator
      • 052 Geographic Classification - 1st indicator (Code source): values: # (Library of Congress Classification), 0 (U.S. Dept. of Defense), 7 (Source specified in subfield $2)
    • Subfield codes
      • $d Populated place name in 052 (Geographic Classification)
      • $y Link text in 856 (Electronic Location and Access)
      • $2 Code source in 052 (Geographic Classification)
      • $3 Materials specified in 050 (Library of Congress Classification Number)
      • $6 Linkage in 050 (Library of Congress Classification Number)
      • $6 Linkage in 654 (Subject Added Entry - Faceted Topical Terms)
  • Changes in content designator field names:
    • Fields
      • 052 Geographic Classification Code
      • 522 Geographic Area Coverage Note
