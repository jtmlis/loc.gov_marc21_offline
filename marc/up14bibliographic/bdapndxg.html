This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Bibliographic Data.

MARC 21 FORMAT FOR BIBLIOGRAPHIC DATA

Update No. 14, April 2012
This page documents the changes contained in the fourtheenth update to the 1999 edition of the MARC 21 Format for Bibliographic Data. Update No. 14 (April 2012) includes changes resulting from proposals that were considered by the MARC 21 community during January 2012.
