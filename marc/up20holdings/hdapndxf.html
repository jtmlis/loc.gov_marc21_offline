This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Holdings Data.

MARC 21 FORMAT FOR HOLDINGS DATA

FORMAT CHANGE LIST

Update No. 20, April 2015
This page documents the changes contained in the twentieth update to the 1999 edition of the MARC 21 Format for Holdings Data. Update No. 20 (April 2015) includes changes resulting from proposals that were considered by the MARC 21 community during January/February 2015.

Format changes in previous updates can be found on the MARC Format Overview page.
