TECHNICAL NOTICE (November 6, 2002)

Format for Authority Data

Format Changes List
Update No. 3 (October 2002)

Network Development and MARC Standards Office
Library of Congress

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Authority Data. It can also be found in Appendix F of the MARC 21 Format for Authority Data.

This page documents the changes contained in the third update to the 1999 edition of the MARC 21 Format for Authority Data. Update No. 3 (October 2002) includes changes resulting from proposals which were considered by the ALA ALCTS/LITA/RUSA Machine-Readable Bibliographic Information Committee (MARBI) at its meetings in 2002 and by the Canadian Committee on MARC (CCM) at its meetings in 2002.

LC Implementation Plans
Subscribers can anticipate receiving MARC records reflecting these changes in all distribution services no earlier than February 6, 2003.

  • New content designator
    • Fields
      • 065 - Other classification number
      • 148 - Heading–Chronological term
      • 448 - See from tracing–Chronological term
      • 548 - See also from tracing–Chronological term
      • 748 - Established heading linking entry–Chronological term
    • Subfield codes
      • $u - Uniform Resource Identifier in 670 (Source data found)
      • $w/1 - Replacement complexity in 7XX (Heading Linking Entry Fields)
    • Indicator
      • 052 - Geographic classification 1st indicator (Code source): value 1 (Department of Defense Classification)
    • Code value
      • Leader - character position 05 (Record status): code o (Obsolete)

  • Obsolete content designator
    • Indicator
      • 052 - Geographic classification 1st indicator (Code source): value 0 (Department of Defense Classification)
