MARC: News and Announcements (November 6, 2002)

TECHNICAL NOTICE (November 6, 2002)

Format for Holdings Data

Format Changes List
Update No. 3 (October 2002)

Network Development and MARC Standards Office
Library of Congress

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Holdings Data. It can also be found in Appendix F of the MARC 21 Format for Holdings Data.

This page documents the changes contained in the third update to the 2000 edition of the MARC 21 Format for Holdings Data. Update No. 3 (October 2002) includes changes resulting from proposals which were considered by the ALA ALCTS/LITA/RUSA Machine-Readable Bibliographic Information Committee (MARBI) at its meetings in 2002 and by the Canadian Committee on MARC (CCM) at its meetings in 2002.

LC Implementation Plans
Subscribers can anticipate receiving MARC records reflecting these changes in all distribution services no earlier than February 6, 2003.

  • New content designators
    • Field
      • 563 - Binding Information
    • Subfield codes
      • $n - Pattern note in 853-855 (Captions and Pattern Fields)
      • $p - Number of pieces per issuance in 853-855 (Captions and Pattern Fields)
    • Code values
      • Field 007 Sound recordings - character position 10 (Kind of material): codes b (Cellulose nitrate); c (Acetate tape with ferrous oxide); g (Glass with lacquer); i (Aluminum with lacquer); r (Paper with lacquer or ferrous oxide); z (Other)
      • Field 008 - character position 07 (Method of acquisition): codes c (Cooperative or consortial purchase); m (Membership); n (Non-library purchase)
      • Field 008 - character position 20 (Lending policy): codes c (Will lend hard copy only); l (Limited lending policy)
      • Fields 853-855 - subfield $y (Regularity pattern): code c (Combined)
      • Fields 853-855 - subfield $y (Regularity pattern): code e (Enumeration)
      • Fields 853-855 - subfield $y (Regularity pattern): code y (Year)

  • Obsolete content designators
    • Code value
      • Field 007 Sound recordings - character position 10 (Kind of material): code n (Not applicable)

  • Changes in content designator names
    • Fields
      • 007 Computer file Electronic resource
      • 008/08-11 Intent to cancel date Expected acquisition end date
    • Code values
      • 007/10 (Sound recording) code a: Lacquered coating
      • 007/10 (Sound recording) code m: Metal and plastic Plastic with metal
      • 007/11 (Computer file) code c: File reproduced from an computer file electronic resource
