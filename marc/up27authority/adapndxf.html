Appendix F - Format Changes

MARC 21 Authority
November 2018

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Authority Data.

MARC 21 FORMAT FOR AUTHORITY DATA

FORMAT CHANGE LIST

Update No. 27, November 2018
This page documents the changes contained in the twenty-seventh update to the 1999 edition of the MARC 21 Format for Authority Data. Update No. 27 (November 2018) includes changes resulting from proposals that were considered by the MARC 21 community since the publication of Update No. 26 (April 2018).

Format changes in previous updates can be found on the MARC Format Overview page.
UPDATE 27 CHANGES
■ Changes in content designator names and definitions:
Subfields
$6 Linkage redescribed in Appendix A (Control Subfields)

■ Fast-track changes:
New Subfield
$2 Number source (NR) in 055 (Library and Archives Canada Call Number)

UPDATE 27 LOCAL PRINT GUIDE

The following sections and fields were created/changed in Update No. 27 and should be printed to keep a print copy of the format up-to-date. Changes appear in red.

  • 055 - Library and Archives Canada Call Number

  • Appendix A - Control Subfields
