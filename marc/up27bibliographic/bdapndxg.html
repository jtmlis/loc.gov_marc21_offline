Appendix G - Format Changes

MARC 21 Bibliographic
November 2018

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Bibliographic Data.

MARC 21 FORMAT FOR BIBLIOGRAPHIC DATA

FORMAT CHANGE LIST

Update No. 27, November 2018
This page documents the changes contained in the twenty-seventh update to the 1999 edition of the MARC 21 Format for Bibliographic Data. Update No. 27 (November 2018) includes changes resulting from proposals that were considered by the MARC 21 community since the publication of Update No. 26 (April 2018).

Format changes in previous updates can be found on the MARC Format Overview page.

UPDATE 27 LOCAL PRINT GUIDE

The following sections and fields were created/changed in Update No. 27 and should be printed to keep a print copy of the format up-to-date. Changes appear in red.

  • Summary Statement of Content Designators

  • 041 - Language Code

  • 25X-28X - Edition, Imprint, Etc. Fields - General Information

  • 251 - Version Information

  • 3XX - Physical Description, Etc. Fields - General Information

  • 341 - Accessibility Content

  • 5XX - Field Notes - General Information

  • 532 - Accessibility Note

  • Appendix A - Control Subfields
