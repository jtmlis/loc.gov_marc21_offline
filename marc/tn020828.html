TECHNICAL NOTICE (August 28, 2002)

Addition to the MARC Code List for Geographic Areas

Network Development and MARC Standards Office
Library of Congress

The following code has been recently approved for use in MARC 21 records as a geographic area code for field 043. The Coral Sea Islands were declared a territory of Australia in 1969. Scattered over 1 million square kilometers of ocean, they are basically uninhabited. The new code will appear in the next edition of the MARC Code List for Geographic Areas.

New Code
Place Name
u-cs Coral Sea Islands

LC Implementation Plans
Subscribers can anticipate receiving MARC records reflecting these changes in all distribution services not earlier than November 29, 2002.
