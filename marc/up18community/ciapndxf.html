Appendix F - Format Changes

MARC 21 Community Information
April 2014

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Community Information.

MARC 21 FORMAT FOR COMMUNITY INFORMATION

FORMAT CHANGE LIST

Update No. 18, April 2014
This page documents the changes contained in the eighteenth update to the 1999 edition of the MARC 21 Format for Community Information. Update No. 18 (April 2014) includes changes resulting from proposals that were considered by the MARC 21 community during January 2014.

UPDATE 18 LOCAL PRINT GUIDE

The following sections and fields were created/changed in Update No. 18 and should be printed to keep a print copy of the format up-to-date. Changes appear in red.

  • X10 - Corporate Names-General Information

  • X11 - Meeting Names-General Information

  • 110 - Primary Name-Corporate

  • 111 - Primary Name-Meeting

  • 610 - Subject Added Entry-Corporate Name

  • 611 - Subject Added Entry-Meeting Name

  • 710 - Added Entry-Corporate Name

  • 711 - Added Entry-Meeting Name
