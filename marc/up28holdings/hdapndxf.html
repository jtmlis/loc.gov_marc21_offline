Appendix F - Format Changes

MARC 21 Holdings
May 2019

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Holdings Data.

MARC 21 FORMAT FOR HOLDINGS DATA

FORMAT CHANGE LIST

Update No. 28, May 2019
This page documents the changes contained in the twenty-eighth update to the 1999 edition of the MARC 21 Format for Holdings Data. Update No. 28 (May 2019) includes changes resulting from proposals that were considered by the MARC 21 community since the publication of Update No. 27 (November 2018).

Format changes in previous updates can be found on the MARC Format Overview page.
UPDATE 28 CHANGES
■ New content designators:
Subfields
$gAvailability date (R) in 506 (Restrictions on Access Note)
$qSupplying agency (NR) in 506 (Restrictions on Access Note)


UPDATE 28 ADDENDUM CHANGES
(JULY 2019)
■ New content designators:
Subfields
$fUse and reproduction rights (R) in 845 (Terms Governing Use and
         Reproduction Note)
$gAvailability date (R) in 845 (Terms Governing Use and Reproduction Note)
$qSupplying agency (NR) in 845 (Terms Governing Use and Reproduction Note)
$2Source of term (NR) in 845 (Terms Governing Use and Reproduction Note)
$7Access status (NR) in 856 (Electronic Location and Access)

■ Withdrawn element previously published in Update 28
Field
540Terms Governing Use and Reproduction Note rescinded. The new field is
           redundant because it is identical to existing field 845 (Terms Governing Use
           and Reproduction Note). The four new subfields in field 845 listed above
           have been defined instead.

UPDATE 28 LOCAL PRINT GUIDE

The following sections and fields were created/changed in Update No. 28 and should be printed to keep a print copy of the format up-to-date. Changes appear in red.

  • 506 - Restrictions on Access Note

  • 845 - Terms Governing Use and Reproduction Note

  • 856 - Electronic Location and Access
