TECHNICAL NOTICE (January 25, 2002)

Format for Holdings Data

Format Changes List
Update No. 2 (October 2001)

Network Development and MARC Standards Office
Library of Congress

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Holdings Data. It can also be found in Appendix F of the MARC 21 Format for Holdings Data.

This page documents the changes contained in the second update to the 2000 edition of the MARC 21 Format for Holdings Data. Update No. 2 (October 2001) includes changes resulting from proposals which were considered by the ALA ALCTS/LITA/RUSA Machine-Readable Bibliographic Information Committee (MARBI) at its meetings in 2001 and by the Canadian Committee on MARC (CCM) at its meetings in 2001.

LC Implementation Plans
Subscribers can anticipate receiving MARC records reflecting these changes in all distribution services no earlier than April 25, 2002.

  • New content designators
    • Coded values
      • Field 853-855 - subfield $w (Frequency): code k (Continuously updated)
      • Field 843 - subfield $7 (Fixed-length data elements of reproduction): code k (Continuously updated)
    • Subfield code
      • $u - Uniform Resource Identifier in 583 (Action note)
