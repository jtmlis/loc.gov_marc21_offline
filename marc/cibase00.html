TECHNICAL NOTICE (July 11, 2000)

Format for Community Information

Format Changes List
Base Text (January 2000)

Network Development and MARC Standards Office
Library of Congress

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Community Information. It can also be found in Appendix F of the MARC 21 Format for Community Information .

This edition of the MARC 21 Format for Community Information supersedes the January 1993 base text and update No. 1 (April 1996) to the USMARC Format for Community Information . It includes changes resulting from proposals which were considered by the ALA ALCTS/LITA/RASD Machine-Readable Bibliographic Information Committee (MARBI) at its meetings from 1997 through January 2000 and by the Canadian Committee on MARC (CCM) at its meetings from 1998 through January 2000.

  • Deleted content designators:
    • Field
      • 004 - Coded Dates Fixed Field [redefined as field 046 (Special Coded Dates)]
    • Subfield codes
      • 700, 710, 711, 730 - $w (Record control number)
      • 856 - $g (Uniform Resource Name) [Incorporated into 856 u]
  • New content designators:
    • Indicators
      • 856 - Electronic Location Access - 2nd indicator (Relationship to Source): values (No information provided), 0 (Electronic location of resource described), 1 (Electronic location of online version of resource described), 2 (Related electronic resource), 8 (No display constant generated)
    • Subfield codes
      • $b - Local GAC code in 043 (Geographic Area Code)
      • $b - Item Number in 080 (Universal Decimal Classification Number)
      • $j - Attribution qualifier in 100 (Primary Name - Personal); 600 (Subject Added Entry - Personal Name); 700 (Added Entry - Personal Name)
      • $v - Form subdivision added in 656 (Index Term - Occupation)
      • $v - Form subdivision added in 657 (Index Term - Function)
      • $x - Common auxiliary subdivision in 080 (Universal Decimal Classification Number)
      • $2 - Source of extension in 043 (Geographic Area Code)
      • $2 - Edition identifier in 080 (Universal Decimal Classification Number)
      • $6 - Linkage in 080 (Universal Decimal Classification Number)
      • $6 - Linkage in 082 (Dewey Decimal Classification Number)
    • Code value
      • Leader - character position 09 (Character coding scheme): codes (MARC-8), a (UCS/Unicode)
  • Changes in content designator field names:
    • Subfield code
      • 856 - $u Uniform Resource Locator Identifier
