MARC: News and Announcements (October 22, 2007)

MARC Code List for Languages

Network Development and MARC Standards Office
Library of Congress

A new 2007 edition of the MARC Code List for Languages is now available online at: www.loc.gov/marc/languages/langhome.html. It may also be accessed from the MARC homepage. The publication is presented in PDF with bookmarks for navigation. The Library's Cataloging Distribution Service will also be publishing a print version of the languages codes and it will be announced when it is available.

This edition incorporates all of the additions and changes since the 2003 edition. These changes had been previously announced and a change list maintained on the MARC web site since the 2003 edition was published. The new edition content was edited by Millicent Wewerka in the Cataloging Policy and Support Office to incorporate all needed changes to code names,group codes languages, and references.

The list is published from a new XML file that is also made available from the site at: www.loc.gov/standards/codelists/languages.xml. (It was formerly published from an SGML file developed in the
late 1990s). The services available from the XML file will be enhanced over the coming months as the other MARC code lists are also released in XML. The 2007 online edition and XML file will be updated on a regular basis.

Questions on the new edition or the XML file should be addressed to Jackie Radebaugh at [email protected].
