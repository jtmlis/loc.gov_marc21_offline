Como Solicitar um Código MARC de Organização

Instruções

Complete separadamente o formulário a seguir para cada uma das organizações que necessitam o código MARC. Se você precisar de mais que três códigos para entidades dentro de uma organização maior (por exemplo, várias escolas de um mesmo distrito escolar), por favor siga as instruções alternativas para códigos múltiplos abaixo. A Biblioteca do Congresso emitirá um código MARC para qualquer organização que necessite de um código, independentemente do seu tamanho. Grandes organizações devem pedir um número separado para divisiões ou unidades subordinadas. Os Códigos não são atribuídos a pessoas, projetos, ou programas. As organizações canadenses podem obter símbolos da biblioteca (códigos MARC) na Divisão de Empréstimo Entre Bibliotecas da Biblioteca Nacional do Canadá (Email: [email protected] ).

Os códigos são usados nos catálogos coletivos para identificar as bibliotecas depositárias e nos registros MARC, para relacionar um registro ou informação específica de um registro a uma organização. Dois campos chave do MARC 21 que requerem um código MARC da organização são os campos 040 (Fonte da Catalogação) e o campo 852 (Localização). Os códigos MARC da organização ("ORG") são identificadores únicos que consistem de um (1) a oito (8) caracteres alfabéticos. A primeira parte do código é um prefixo geográfico que identifica a localização (estado e cidade, ou, país e cidade) da organização. A última parte do código usa letras do nome da organização para completar o código. Enquanto os códigos são independentes ao determinar a exclusividade, eles geralmente são gravados em combinações de letras maiúsculas e minúsculas que ajudam a identificar os componentes do código. Por exemplo: "BlRjBN" é o código MARC para a Biblioteca Nacional, em Rio de Janeiro.

Por favor forneça o maior número possível de informações. Elas serão carregadas diretamente numa base de Dados de códigos MARC usada para publicar a lista de códigos das organizações. Use a tecla [TAB] para mudar para o próximo quadro em branco. Siga as instruções para cada uma das informações abaixo. Qualquer informação formatada incorretamente terá que ser corrigida manualmente, o que poderá causar atrasos na atribuição de novos códigos. Após preencher este formulário, pressione o botão [SUBMETER A SOLICITAÇÃO].

Obrigado por solicitar um código de organização MARC.
