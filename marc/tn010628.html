TECHNICAL NOTICE (June 28, 2001)

Test Files Available Containing New MARC 21 Data Elements

Network Development and MARC Standards Office
Library of Congress

As previously announced new data elements from recent updates to the MARC 21 bibliographic and authority formats will become valid in LC's system no earlier than August 15, 2001. In order to assist subscribers in accommodating the new elements, test files containing the new elements are available via anonymous FTP at:

Host: ftp.loc.gov
Login: anonymous
Password: <your email address>
Directory: pub
Subdirectory: cds/lccn
Filenames: tesnewauth (authority test file) tesnewbib (bibliographic test file)

Questions regarding the available test files should be directed to:
