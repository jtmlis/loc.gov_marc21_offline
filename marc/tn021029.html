TECHNICAL NOTICE (October 29, 2002 )

Additions to MARC Code List for Languages

Network Development and MARC Standards Office
Library of Congress

The following codes have been approved for use in the international language code standard, ISO 639-2 (Codes for the Representation of Names of Languages--Part 2: alpha-3 code) and are likewise being added to the MARC Code List for Languages.

 
New code Language name Previously coded
iii Sichuan Yi n/a
lim Limburgish n/a
nap Neapolitan Italian ita

 

LC Implementation Plans
Subscribers can anticipate receiving MARC records reflecting these changes in all distribution services not earlier than January 29, 2003.
