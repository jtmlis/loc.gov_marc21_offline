Appendix G - Format Changes

MARC 21 Bibliographic
October 2007
This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Bibliographic Data.

MARC 21 FORMAT FOR BIBLIOGRAPHIC DATA

FORMAT CHANGE LIST

Update No. 8, October 2007
This page documents the changes contained in the eighth update to the 1999 edition of the MARC 21 Format for Bibliographic Data. Update No. 8 (October 2007) includes changes resulting from proposals that were considered by the MARC 21 community during 2007.
