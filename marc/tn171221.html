TECHNICAL NOTICE (December 21, 2017)

Changes to MARC Code List for Languages

Network Development and MARC Standards Office
Library of Congress

As a result of a formal request from the National Library of Montenegro and Montenegro's national standards body to the ISO 639 Joint Advisory Committee, the following new code has been approved for use in the international language code standard, ISO 639-2 (Codes for the Representation of Names of Languages--Part 2: alpha-3 code) and is consequently also being added to the MARC Code List for Languages.

New code Language name Previously coded
cnr Montenegrin srp

The language code “srp” will no longer be used for the variant of the Serbo-Croatian language known as Montenegrin, that is spoken in the sovereign state of Montenegro. Code "srp" will continue to be used for the variant of Serbo-Croatian spoken in Serbia and other areas.  In the MARC Code List for Languages, Montenegrin will be listed as follows:

Montenegrin [cnr]

Subscribers can anticipate receiving MARC records reflecting these changes in all distribution services not earlier than February 21, 2018. Questions regarding the language codes should be directed to:

Network Development & MARC Standards Office
Library of Congress
Washington, DC 20540-4305
(202) 707-6237
