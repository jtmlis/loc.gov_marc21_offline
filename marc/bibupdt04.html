TECHNICAL NOTICE ( January 23, 2004 )

Format for Bibliographic Data

Format Changes List
Update No. 4 (October 2003)

Network Development and MARC Standards Office
Library of Congress

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Bibliographic Data. It can also be found in Appendix G of the MARC 21 Format for Bibliographic Data.

This page documents the changes contained in the fourth update to the 1999 edition of the MARC 21 Format for Bibliographic Data. Update No. 4 (October 2003) includes changes resulting from proposals which were considered by the ALA ALCTS/LITA/RUSA Machine-Readable Bibliographic Information Committee (MARBI) at its meetings in 2003, the Canadian Committee on MARC (CCM) at its 2003 meetings and by the Book Industry Communications/Bibliographic Standards Technical Subgroup (BIC/BSTS) at its meetings in 2003.

LC Implementation Plans
Subscribers can anticipate receiving MARC records reflecting these changes in all distribution services no earlier than April 23, 2004.

  • New content designators:
    • Appendix
      • Appendix I: Organization Code Sources
    • Fields
      • 365 - Trade Price
      • 366 - Trade Availability Information
    • Subfield codes
      • $i - Display text in 538 (System details note)
      • $q - Enumeration and first page in 773 (Host item entry)
      • $q - Format of the digital image in 352 (Digital graphic representation)
      • $u - Uniform Resource Identifier in 538 (System details note)
      • $2 - Source in 022 (International Standard Serial Number)
      • $3 - Materials specified in 538 (System details note)
  • Obsolete content designators:
    • Character positions
      • CR 006/03 (ISSN Center)
      • CR 008/20 (ISSN Center)
  • Changes in content designator names:
    • Subfield codes
      • $a in 044 (Country of Publishing/producing Entity Code): MARC Country of Publishing/producing Entity Code
      • $c in 044 (Country of Publishing/producing Entity Code): ISO Country Code
      • $2 in 072 (Subject Category Code): Code Source
    • Indicator value
      • 2nd indicator (Code source) in 072: Code Source Specified in Subfield $2
