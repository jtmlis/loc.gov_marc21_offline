Appendix F - Format Changes

MARC 21 Authority
September 2011

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Authority Data.

MARC 21 FORMAT FOR AUTHORITY DATA

FORMAT CHANGE LIST

Update No. 13, September 2011
This page documents the changes contained in the thirteenth update to the 1999 edition of the MARC 21 Format for Authority Data. Update No. 13 (September 2011) includes changes resulting from proposals that were considered by the MARC 21 community during January and June 2011.
UPDATE 13 CHANGES
■ New content designators:
Indicators
0821st indicator (Type of edition):
7 - Other edition specified in subfield $2
0831st indicator (Type of edition):
7 - Other edition specified in subfield $2
Subfields
$d Thematic index code (NR) in 383 (Numeric Designation of Musical Work)
$e Publisher associated with opus number (NR) in 383 (Numeric Designation
          of Musical Work)
$l Language term (R) in 377 (Associated Language)
$0 Authority record control number or standard number (R) in 034 (Coded
          Cartographic Mathematical Data)
$0 Authority record control number or standard number (R) in 043 (Geographic
          Area Code)
$2 Source (NR) in 383 (Numeric Designation of Musical Work)
Fields
368 Other Corporate Body Attributes (R)
378 Fuller Form of Personal Name (NR)
■ Changes in content designator names:
Subfields
$aAffiliation changed to Associated group and redefined in 373 (Associated Group)
Fields
373Affiliation changed to Associated Group

UPDATE 13 LOCAL PRINT GUIDE

The following sections and fields were created/changed in Update No. 13 and should be printed to keep a print copy of the format up-to-date. Changes appear in red.

  • 034 - Coded Cartographic Mathematical Data

  • 043 - Geographic Area Code

  • 082 - Dewey Decimal Call Number

  • 083 - Dewey Decimal Classification Number

  • Headings - General Information

  • 368 - Other Corporate Body Attributes

  • 373 - Associated Group

  • 377 - Associated Language

  • 378 - Fuller Form of Personal Name

  • 383 - Numeric Designation of Musical Work
