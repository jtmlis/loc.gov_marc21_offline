Appendix F - Format Changes

MARC 21 Community Information
October 2014

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Community Information.

MARC 21 FORMAT FOR COMMUNITY INFORMATION

FORMAT CHANGE LIST

Update No. 19, October 2014
This page documents the changes contained in the nineteenth update to the 1999 edition of the MARC 21 Format for Community Information. Update No. 19 (October 2014) includes changes resulting from proposals that were considered by the MARC 21 community during June 2014.

Format changes in previous updates can be found on the MARC Format Overview page.
