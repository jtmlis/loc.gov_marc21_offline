Appendix G - Format Changes

MARC 21 Bibliographic
November 2019

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Bibliographic Data.

MARC 21 FORMAT FOR BIBLIOGRAPHIC DATA

FORMAT CHANGE LIST

Update No. 29, November 2019
This page documents the changes contained in the twenty-ninth update to the 1999 edition of the MARC 21 Format for Bibliographic Data. Update No. 29 (November 2019) includes changes resulting from proposals that were considered by the MARC 21 community since the publication of Update No. 28 (May 2019).

Format changes in previous updates can be found on the MARC Format Overview page.
UPDATE 29 CHANGES
■ New content designators:
Field
688Subject Added Entry-Type of Entity Unspecified (R)
Subfields
$i Language code of intertitles (R) in 041 (Language Code)
$t Language code of accompanying transcripts for audiovisual materials (R)
         in 041 (Language Code)

■ Changes in content designator names and definitions:
Subfield
$g Language code of accompanying material other than librettos renamed
         Language code of accompanying material other than librettos and
         transcripts in 041 (Language Code)

■ Fast-track change:
New Subfield
$g Miscellaneous information (R) in 751 (Added Entry-Geographic Name)

UPDATE 29 LOCAL PRINT GUIDE

The following sections and fields were created/changed in Update No. 29 and should be printed to keep a print copy of the format up-to-date. Changes appear in red.

  • Summary Statement of Content Designators

  • 041 - Language Code

  • 6XX - Subject Access Fields - General Information

  • 688 - Subject Added Entry - Type of Entity Unspecified

  • 751 - Added Entry - Georgraphic Name
