Appendix G - Format Changes

MARC 21 Bibliographic
May 2017

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Bibliographic Data.

MARC 21 FORMAT FOR BIBLIOGRAPHIC DATA

FORMAT CHANGE LIST

Update No. 24, May 2017
This page documents the changes contained in the twenty-fourth update to the 1999 edition of the MARC 21 Format for Bibliographic Data. Update No. 24 (May 2017) includes changes resulting from proposals that were considered by the MARC 21 community during January 2017.

Format changes in previous updates can be found on the MARC Format Overview page.
