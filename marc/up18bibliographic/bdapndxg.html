Appendix G - Format Changes

MARC 21 Bibliographic
April 2014

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Bibliographic Data.

MARC 21 FORMAT FOR BIBLIOGRAPHIC DATA

FORMAT CHANGE LIST

Update No. 18, April 2014
This page documents the changes contained in the eighteenth update to the 1999 edition of the MARC 21 Format for Bibliographic Data. Update No. 18 (April 2014) includes changes resulting from proposals that were considered by the MARC 21 community during January 2014.
UPDATE 18 CHANGES
■ New content designators:
Indicators
5881st indicator (Display constant controller):
# - No information provided
0 - Source of description
1 - Latest issue consulted
■ Changes in content designator names and definitions:
Codes
Code q - Quadraphonic renamed Quadraphonic, multichannel, or surround
                            and redescribed in Sound Recording 007/04 (Configuration of playback
                            channels).
Subfields
$fTransmission speed renamed Encoded bitrate in 347 (Digital File
         Characteristics)
$cLocation of meeting changed to repeatable in:
         110 (Main Entry-Corporate Name),
         111 (Main Entry-Meeting Name),
         610 (Subject Added Entry-Corporate Name),
         611 (Subject Added Entry-Meeting Name),
         710 (Added Entry-Corporate Name),
         711 (Added Entry-Meeting Name),
         810 (Series Added Entry-Corporate Name),
         811 (Series Added Entry-Meeting Name).

UPDATE 18 LOCAL PRINT GUIDE

The following sections and fields were created/changed in Update No. 18 and should be printed to keep a print copy of the format up-to-date. Changes appear in red.

  • 007 - Sound Recording

  • X10 - Corporate Names-General Information

  • X11 - Meeting Names-General Information

  • 110 - Main Entry-Corporate Name

  • 111 - Main Entry-Meeting Name

  • 347 - Digital File Characteristics

  • 588 - Source of Description Note

  • 610 - Subject Added Entry-Corporate Name

  • 611 - Subject Added Entry-Meeting Name

  • 710 - Added Entry-Corporate Name

  • 711 - Added Entry-Meeting Name

  • 810 - Series Added Entry-Corporate Name

  • 811 - Series Added Entry-Meeting Name
