TECHNICAL NOTICE (January 25, 2002)

Format for Community Information

Format Changes List
Update No. 2 (October 2001)

Network Development and MARC Standards Office
Library of Congress

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Community Information. It can also be found in Appendix F of the MARC 21 Format for Community Information.

This page documents the changes contained in the second update to the 2000 edition of the MARC 21 Format for Community Information. Update No. 2 (October 2001) includes changes resulting from proposals which were considered by the ALA ALCTS/LITA/RUSA Machine-Readable Bibliographic Information Committee (MARBI) at its meetings in 2001 and by the Canadian Committee on MARC (CCM) at its meetings in 2001.

LC Implementation Plans
Subscribers can anticipate receiving MARC records reflecting these changes in all distribution services no earlier than April 14, 2002.

  • New content designators
    • Indicator
      • 041 - Language code - 2nd indicator (Source of code): values # (MARC language code); 7 (Source specified in subfield $2)
    • Subfield codes
      • $c - ISO code in 043 (Geographic area code)
      • $2 - Source of code in 041 (Language code)
  • Changes in repeatability:
    • Field
      • 041 - Language code : changed from Not repeatable (NR) to Repeatable (R)
    • Subfield codes
      • $a - Language code associated with community information entity in 041 (Language code): changed from Not repeatable (NR) to Repeatable (R)
      • $b - Language code of supertitle or subtitle in 041 (Language code): changed from Not repeatable (NR) to Repeatable (R)
      • $h - Language code of original in 041 (Language code): changed from Not repeatable (NR) to Repeatable (R)
