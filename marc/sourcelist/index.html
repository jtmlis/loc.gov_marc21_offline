MARC Field Index to Source Code Usage

Formerly named MARC Code Lists for Relators, Sources, Description Conventions which included the MARC Relator Code value list. The relator code value list part has moved to MARC Code List for Relators.

Many fields in the MARC Format contain data that may come from controlled lists of terms, codes, or identifiers (e.g., subject terms, language codes, standard identifiers) or are constructed according to specific rules (e.g., RDA) or schemes (e.g., DDC). The lists, rules, and schemes are generally referred to as the source of the data. The following fields contain subfields where the source of the controlled data within the field may be specified. The field name links to the pages that specify the codes for the sources.

In a few cases the link is directly to a table that contains the values for the field.
