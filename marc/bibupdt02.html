MARC: News and Announcements (January 25, 2002)

TECHNICAL NOTICE (January 25, 2002)

Format for Bibliographic Data

Format Changes List
Update No. 2 (October 2001)

Network Development and MARC Standards Office
Library of Congress

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Bibliographic Data. It can also be found in Appendix G of the MARC 21 Format for Bibliographic Data.

This page documents the changes contained in the second update to the 1999 edition of the MARC 21 Format for Bibliographic Data. Update No. 2 (October 2001) includes changes resulting from proposals which were considered by the ALA ALCTS/LITA/RUSA Machine-Readable Bibliographic Information Committee (MARBI) at its meetings in 2001 and by the Canadian Committee on MARC (CCM) at its meetings in 2001.

LC Implementation Plans
Subscribers can anticipate receiving MARC records reflecting these changes in all distribution services no earlier than April 25, 2002.

  • New content designators:
    • Field
      • 887 - Non-MARC information field
    • Indicators
      • 041 - Language code - 2nd indicator (Source of code): values # (MARC language code); 7 (Source specified in subfield $2)
      • 260 - Publication, distribution, etc. (Imprint) - 1st indicator (Sequence of publishing statements): values # (Not applicable/No information provided/Earliest available publisher); 2 (Intervening publisher); 3 (Current/latest publisher)
    • Subfield codes
      • $c - ISO code in 043 (Geographic area code)
      • $c - Taxonomic category in 754 (Added entry--taxonomic identification)
      • $d - Common or alternative name in 754 (Added entry--taxonomic identification)
      • $x - Non-public note in 754 (Added entry--taxonomic identification)
      • $z - Public note in 754 (Added entry--taxonomic identification)
      • $2 - Source in 015 (National bibliography number)
      • $2 - Source in 017 (Copyright or legal deposit number)
      • $2 - Source of code in 041 (Language code)
      • $8 - Field link and sequence number in 018 (Copyright article-fee code)
    • Code values
      • Leader - character position 07 (Bibliographic level): code i (Integrating resource)
      • Field 006 - Serials - character position 01 (Frequency): code k (Continuously updated)
      • Field 006 - Serials - character position 04 (Type of continuing resource): codes d (Updating database); l (Updating loose-leaf); w (Updating Web site)
      • Field 006 - Serials - character position 17 (Entry convention): code 2 (Integrating entry)
      • Field 007 - Videorecordings - character position 04 (Videorecording format): code v (DVD)
      • Field 008 - Serials - character position 18 (Frequency): code k (Continuously updated)
      • Field 008 - Serials - character position 21 (Type of continuing resource): codes d (Updating database); l (Updating loose-leaf); w (Updating Web site)
      • Field 008 - Serials - character position 34 (Entry convention): code 2 (Integrating entry)
      • Field 533 - subfield $7 (Fixed-length data elements of reproduction): code k (Continuously updated)

  • Changes in repeatability:
    • Fields
      • 015 - National bibliography number : changed from Not repeatable (NR) to Repeatable (R)
      • 041 - Language code : changed from Not repeatable (NR) to Repeatable (R)
      • 260 - Publication, distribution, etc. (Imprint) : changed from Not repeatable (NR) to Repeatable
      • 508 - Creation/production credits note : changed from Not repeatable (NR) to Repeatable (R)
    • Subfield codes
      • $a - Language code of text/sound track or separate title in 041 (Language code): changed from Not repeatable (NR) to Repeatable (R)
      • $b - Language code of summary or abstract/overprinted title or subtitle in 041 (Language code): changed from Not repeatable (NR) to Repeatable (R)
      • $d - Language code of sung or spoken text in 041 (Language code): changed from Not repeatable (NR) to Repeatable (R)
      • $e - Language code of librettos in 041 (Language code): changed from Not repeatable (NR) to Repeatable (R)
      • $f - Language code of table of contents in 041 (Language code): changed from Not repeatable (NR) to Repeatable (R)
      • $g - Language code of accompanying material other than librettos in 041 (Language code): changed from Not repeatable (NR) to Repeatable (R)

  • Changes in content designator names:
    • Fields
      • 006/04 (Serials) - Type of continuing resource Type of serial
      • 006/17 (Serials) - Entry convention Successive/latest entry
      • 008/21 (Serials) - Type of continuing resource Type of serial
      • 008/34 (Serials) - Entry convention Successive/latest entry
      • 017 - Copyright Registration or Legal Deposit Number
      • 055 - Call Numbers/ Classification Numbers Assigned in Canada
      • 082 - Dewey Decimal Call Classification Number
      • 084 - Other Call Classification Number
      • 086 - Government Document Call Classification Number
      • 772 - Supplement Parent Record Entry
    • Subfield codes
      • $a in 754 (Added entry--taxonomic identification) Taxonomic name /taxonomic hierarchical category
      • $b in 017 (Copyright or legal deposit number) Assigning agency Source (assigning agency)
      • $c in 044 (Country of publishing/producing entity code) ISO subentity code
    • Coded value
      • 007/04 (Videorecording) code g - Laserdisc Laser optical (Reflective) videodisc
