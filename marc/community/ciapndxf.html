Appendix F - Format Changes

MARC 21 Community Information
December 2020

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Community Information.

MARC 21 FORMAT FOR COMMUNITY INFORMATION

FORMAT CHANGE LIST

Update No. 31, December 2020
This page documents the changes contained in the thirty-first update to the 1999 edition of the MARC 21 Format for Community Information. Update No. 31 (December 2020) includes changes resulting from proposals that were considered by the MARC 21 community since the publication of Update No. 30 (May 2020).

Format changes in previous updates can be found on the MARC Format Overview page.
UPDATE 31 CHANGES
■ Obsolete content designators:
Subfields

The following subfields were made obsolete in field 856 (Electronic Location and Access):

$bAccess number (R)
$hProcessor of Request (NR)
$i Instruction (R)
$j Bits per second (NR)
$kPassword (NR)
$l Logon (NR)
$nName of location of host (NR)
$rSettings (NR)
$tTerminal emulation (R)

■ Fast-track change:
Field
043Geographic Area Code changed to repeatable

UPDATE 31 LOCAL PRINT GUIDE

The following sections and fields were created/changed in Update No. 31 and should be printed to keep a print copy of the format up-to-date. Changes appear in red.

  • 01X-08X - Numbers and Codes - General Information

  • 043 - Geographic Area Code

  • 856 - Electronic Location and Access
