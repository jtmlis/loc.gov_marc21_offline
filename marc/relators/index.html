MARC Code List for Relators

NOTE: The MARC Code Lists for Relators, Sources, Description Conventions have been reorganized. Relator terms and codes can be accessed on this page (below). The source code list parts can now be accessed at Source Codes for Vocabularies, Rules, and Schemes.
List identifier: marcrelator

Introduction

The purpose of this list of relator terms and associated codes is to allow the relationship between an agent and a resource to be designated in bibliographic records. The relator code list is in two sections:

Term Sequence - A list of standard relator terms with codes for terms, references from unused terms, and descriptions/definitions of term concepts.

Code Sequence - A list of valid and obsolete relator codes with their associated term.  Reference are not provided in the code sequence (except for obsolete codes).

Alternative Access

The terms and codes on this list may also be accessed via the Library of Congress Linked Data Service (LDS) at id.loc.gov/vocabulary/relators.  Download of terms or of the complete list in various formats (RDF, JSON, MADS, SKOS, etc.) are available from that site.   

Code Structure

The relator codes are three-character lowercase alphabetic strings. In most cases they consist of the first letter of the first word of the relator term followed by two additional letters derived from the word(s) making up the relator term.

List Maintenance

The Library of Congress is the maintenance agency for this value list.
