TECHNICAL NOTICE (March 13, 2003)

Additions to the MARC Code Lists for Relators, Sources, Description Conventions

Network Development and MARC Standards Office
Library of Congress

The codes listed below have been recently approved for use in MARC 21 records. They include 1 relator code, 1 classification source code, 1 subject source code, and 1 format source code. These new codes will be added to the online MARC Codes Lists for Relators, Sources, Description Conventions. The codes should not be used in exchange records until after May 11, 2003. This 60-day waiting period is required to provide MARC 21 implementers with time to include newly defined codes in any validation tables they may apply to the MARC fields where these codes are used.

  • MARC Relator Codes
    Changes:
    rpy - Responsible party (subfield $4 in name access fields)

  • MARC Classification Scheme Sources
    Changes:
    rubbks - Bibliotechno-bibliograficheskaíà klassifikatsiíà: Srednye tablitsy (subfield $2 in bibliographic field 084 and authority field 065)

  • MARC Term, Name, Title Sources
    Changes:
    sipri - SIPRI library thesaurus (subfield $2 in Bibliographic and Community Information records in fields 600-651, 655-658)

  • MARC Format Sources
    Changes:
    mods- Metadata object description schema (subfield $2 in Bibliographic records in field 887)

The Library of Congress, Network Development and MARC Standards Office provides information about existing and newly assigned MARC codes on its Web site (www.loc.gov/marc/) as well as notices such as these to subscribers to its "MARC" electronic list ([email protected]).
