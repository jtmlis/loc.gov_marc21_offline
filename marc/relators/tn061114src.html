TECHNICAL NOTICE (November 14, 2006)

Addition to the MARC Code Lists for Relators, Sources, Description Conventions

Network Development and MARC Standards Office
Library of Congress

The codes listed below have been recently approved for use in MARC 21 records. The codes will be added to the online MARC Code Lists for Relators, Sources, Description Conventions.

The codes should not be used in exchange records until after January 14, 2007. This 60-day waiting period is required to provide MARC 21 implementers time to include newly defined codes in any validation tables they may apply to the MARC fields where the codes are used.

Other Sources

The following code is for use in subfield $2 in field 506 (Restrictions on Access Note) in Bibliographic and Holdings records.

  • Addition:
    • star - Standardized terminology for access restriction (DLF/OCLC Registry of Digital Masters Working Group) http://purl.oclc.org/NET/506Vocabulary [use only after January 14, 2007] [PDF: 104 KB] (requires the free Acrobat Reader to view)

Term, Name, Title Sources

The following codes are for use in subfield $2 in Bibliographic and Community Information records in fields 600-651 and 655-657 and in subfield $f in field 040 (Cataloging Source) in Authority records.

  • Additions:
    • aucsh - Qâ'imat ru'ûs mawdû'ât al-fahras al-'Arabîyah al-mowahad = Arabic Union Catalog Subject Headings [use only after January 14, 2007]
    • spines - Tesauro SPINES: un vocabulario controlado y estructurado para el tratamiento de información sobre ciencia y tecnología para el desarrollo (Madrid: UNESCO, ICYT) [use only after January 14, 2007]

The following code is for use in subfield $2 in Bibliographic records in field 752 (Added Entry - Hierarchical Place Name).

  • Addition:
    • blnpn - British Library newspaper place names (London: British Library) [use only after January 14, 2007]

The Library of Congress, Network Development and MARC Standards Office provides information about existing and newly assigned MARC codes on its Web site (www.loc.gov/marc/) as well as notices such as these to subscribers to its MARC listserv. ([email protected]).
