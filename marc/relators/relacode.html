MARC Code List for Relators

Code Sequence

List identifier: marcrelator

Arrangement of the List

In the Code Sequence list, the relator terms and their codes are listed alphabetically by the code. Only the code and the term are provided in this list. For example:

            art       Artist

Discontinued codes are included in this list. They are identified by a hyphen preceding the code. For example:

            -grt       Graphic technician

Definitions, tracings or references for variants of terms and use notes are in the Term Sequence list.
