TECHNICAL NOTICE (July 30, 2009)

Additions to the MARC Code Lists for Relators, Sources, Description Conventions

Network Development and MARC Standards Office
Library of Congress

The codes listed below have been recently approved for use in MARC 21 records. The codes will be added to MARC Code Lists for Relators, Sources, Description Conventions.

The codes should not be used in exchange records until after Spetember 30, 2009. This 60-day waiting period is required to provide MARC 21 implementers time to include newly-defined codes in any validation tables they may apply to the MARC fields where the codes are used.

Classification Code Sources

The following code is for use in subfield $2 in field 084 (Other Classification Number) in Bibliographic and Community Information records, in subfield $2 in field 084 (Classification Scheme and Edition) in Classification records, in subfield $2 in field 065 (Other Classification Number) in Authority records and in subfield $2 in field 852 (Location) in Bibliographic and Holdings records.

  • Addition:
    • bkl - Basisklassifikation [use only after September 30, 2009]

A complete printable edition of Basisklassifikation is available online. [In PDF format; requires the use of a PDF reader.]

Other Sources

The following code is for use in subfield $2 in field 024 (Other Standard Identifier) in in Bibliographic and Holdings records

  • Addition:
    • nipo - Número de Identificación de las Publicaciones Oficiales [use only after September 30, 2009]

The Library of Congress, Network Development and MARC Standards Office provides information about existing and newly assigned MARC codes on its Web site (www.loc.gov/marc/) as well as notices such as these to subscribers to its MARC listserv ([email protected]).
