TECHNICAL NOTICE (June 20, 2003)

Additions to the MARC Code Lists for Relators, Sources, Description Conventions

Network Development and MARC Standards Office
Library of Congress

The codes listed below have been recently approved for use in MARC 21 records. They include 2 relator codes, 1 subject category code, 1 classification source code, 4 subject source codes, 1 format source code, 3 standard number source codes, and 2 language code source codes. These new codes will be added to the online MARC Codes Lists for Relators, Sources, Description Conventions. The codes should not be used in exchange records until after August 17, 2003. This 60-day waiting period is required to provide MARC 21 implementers with time to include newly defined codes in any validation tables they may apply to the MARC fields where these codes are used.

  • MARC Relator Codes
    Changes:
    fpy - First party (subfield $4 in name access fields)
    spy - Second party (subfield $4 in name access fields)

  • MARC Category Code Sources
    Changes:
    lcsh - Library of Congress subject headings (subfield $z in authorities field 073)
  • MARC Classification Scheme Sources
    Changes:
    celex - CELEX document number (subfield $2 in bibliographic field 086 and authority fields 086 and 087)

  • MARC Term, Name, Title Sources
    Changes:
    galestne - Gale Group subject thesaurus and named entity vocabulary (subfield $2 in Bibliographic and Community Information records in fields 600-651, 655-658)
    hapi - HAPI thesaurus and named authority, 1970-2000 (subfield $2 in Bibliographic and Community Information records in field 600-651, 655-658)
    opms - Opetusministeriön asaisanasto (subfield $2 in Bibliographic and Community Information records in field 600-651, 655-658)
    sgp - Svenska genrebeteckningar för periodika (subfield $2 in Bibliographic and Community Information records in field 655)

  • MARC Format Sources
    Changes:
    dc- Dublin core (subfield $2 in Bibliographic records in field 887)

  • Other Sources
    Changes:
    isan - International standard audiovisual number (subfield $2 in Bibliographic records in field 024)
    istc - International standard text code (subfield $2 in Bibliographic records in field 024)
    iswc - International standard musical work code (subfield $2 in Bibliographic records in field 024)
    iso639-1 - Codes for the representation of names of languages--Part 1: alpha-2 code (ISO 639-1:2002) (subfield $2 in Bibliographic records in field 041)
    knia - Kody naimenovanii íàzykov: GOST 7.75-97 (subfield $2 in Bibliographic records in field 041)

The Library of Congress, Network Development and MARC Standards Office provides information about existing and newly assigned MARC codes on its Web site (www.loc.gov/marc/) as well as notices such as these to subscribers to its "MARC" electronic list ([email protected]).
