Appendix G - Format Changes

MARC 21 Bibliographic
September 2015

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Bibliographic Data.

MARC 21 FORMAT FOR BIBLIOGRAPHIC DATA

FORMAT CHANGE LIST

Update No. 21, September 2015
This page documents the changes contained in the twenty-first update to the 1999 edition of the MARC 21 Format for Bibliographic Data. Update No. 21 (September 2015) includes changes resulting from proposals that were considered by the MARC 21 community during June 2015.

Format changes in previous updates can be found on the MARC Format Overview page.
UPDATE 21 CHANGES
■ New content designators:
Subfields

$0 Authority record control number or standard number (R) in 337 (Media Type)

UPDATE 21 LOCAL PRINT GUIDE

The following sections and fields were created/changed in Update No. 21 and should be printed to keep a print copy of the format up-to-date. Changes appear in red.

  • Summary Statement of Content Designators

  • Physical Description, Etc. Fields - General Information

  • 336 - Content Type

  • 337 - Media Type

  • 338 - Carrier Type

  • 348 - Format of Notated Music

The following fields do not have technical changes but contain editorial modifications (in red):

  • 344 - Sound Characteristics

  • 345 - Projection Characteristics of Moving Image

  • 346 - Video Characteristics

  • 347 - Digital File Characteristics
