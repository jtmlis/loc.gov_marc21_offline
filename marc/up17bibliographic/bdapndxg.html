Appendix G - Format Changes

MARC 21 Bibliographic
September 2013

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Bibliographic Data.

MARC 21 FORMAT FOR BIBLIOGRAPHIC DATA

FORMAT CHANGE LIST

Update No. 17, September 2013
This page documents the changes contained in the seventeenth update to the 1999 edition of the MARC 21 Format for Bibliographic Data. Update No. 17 (September 2013) includes changes resulting from proposals that were considered by the MARC 21 community during June 2013.
UPDATE 17 CHANGES
■ New content designators:
Subfields
$q Qualifying information (R) in 015 (National Bibliography Number)
$q Qualifying information (R) in 020 (International Standard Book Number)
$q Qualifying information (R) in 024 (Other Standard Identifier)
$q Qualifying information (R) in 027 (Standard Technical Report Number)
$7 Control subfield (NR) in 800 (Series Added Entry-Personal Name)
$7 Control subfield (NR) in 810 (Series Added Entry-Corporate Name)
$7 Control subfield (NR) in 811 (Series Added Entry-Meeting Name)
$7 Control subfield (NR) in 830 (Series Added Entry-Uniform Title)

UPDATE 17 LOCAL PRINT GUIDE

The following sections and fields were created/changed in Update No. 17 and should be printed to keep a print copy of the format up-to-date. Changes appear in red.

  • 015 - National Bibliography Number

  • 020 - International Standard Book Number

  • 024 - Other Standard Identifier

  • 027 - Standard Technical Report

  • X00 - Personal Names-General Information

  • X10 - Corporate Names-General Information

  • X11 - Meeting Names-General Information

  • X30 - Uniform Titles-General Information

  • 800 - Series Added Entry-Personal Name

  • 810 - Series Added Entry-Corporate Name

  • 811 - Series Added Entry-Meeting Name

  • 830 - Series Added Entry-Uniform Title
