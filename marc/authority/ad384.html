FIELD DEFINITION AND SCOPE

The set of pitch relationships that establishes a single pitch class as a tonal centre for a musical work or expression. May be used to differentiate a musical work or expression from another with the same title.

GUIDELINES FOR APPLYING CONTENT DESIGNATORS

■ INDICATORS

First Indicator - Key type

# - Relationship to original unknown

The relationship of the specified key to the original key in which the musical composition was written.

0 - Original key
1 - Transposed key

A different key from that of the original work.

Second Indicator - Undefined

Undefined and contains a blank (#).

■ SUBFIELD CODES

$a - Key

The pitch name and the mode (e.g., major or minor).
