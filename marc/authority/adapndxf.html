Appendix F - Format Changes

MARC 21 Authority
December 2020

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Authority Data.

MARC 21 FORMAT FOR AUTHORITY DATA

FORMAT CHANGE LIST

Update No. 31, December 2020
This page documents the changes contained in the thirty-first update to the 1999 edition of the MARC 21 Format for Authority Data. Update No. 31 (December 2020) includes changes resulting from proposals that were considered by the MARC 21 community since the publication of Update No. 30 (May 2020).

Format changes in previous updates can be found on the MARC Format Overview page.
UPDATE 31 CHANGES
■ New content designator:
Fields
335Extension Plan (R)

■ Obsolete content designators:
Subfields

The following subfields were made obsolete in field 856 (Electronic Location and Access):

$bAccess number (R)
$hProcessor of Request (NR)
$i Instruction (R)
$j Bits per second (NR)
$kPassword (NR)
$l Logon (NR)
$nName of location of host (NR)
$rSettings (NR)
$tTerminal emulation (R)

■ Fast-track changes:
Field
043Geographic Area Code changed to repeatable
New Subfields
$0 Authority record control number or standard number (R) in 375 (Gender)
$0 Authority record control number or standard number (R) in 384 (Key)
$1 Real World Object URI (R) in 375 (Gender)
$1 Real World Object URI (R) in 384 (Key)

UPDATE 31 LOCAL PRINT GUIDE

The following sections and fields were created/changed in Update No. 31 and should be printed to keep a print copy of the format up-to-date. Changes appear in red.

  • Summary Statement of Content Designators

  • 01X-09X - Numbers and Codes - General Information

  • 043 - Geographic Area Code

  • 1XX, 3XX - Heading Information Fields

  • 335 - Extension Plan

  • 375 - Gender

  • 384 - Key

  • 856 - Electronic Location and Access
