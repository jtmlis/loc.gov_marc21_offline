TECHNICAL NOTICE (January 19, 2005)

Additions to MARC Code List for Languages

Network Development and MARC Standards Office
Library of Congress

The following codes have been approved for use in the international language code standard, ISO 639-2 (Codes for the Representation of Names of Languages--Part 2: alpha-3 code) and are also being added to the MARC Code List for Languages.

 
New code Language name Previously coded
csb Kashubian sla (Slavic (Other))
fil Filipino  
krc Karachay-Balkar tut (Altaic (Other))
mdf Moksha fiu (Finno-Ugrian (Other))
mwl Mirandese  
myv Erzya fiu (Finno-Ugrian (Other))
nwc Newari, Old  
scn Sicilian Italian ita (Italian)
srn Sranan cpe (Creoles and Pidgins, English-based (Other)
tlh
 Klingon (Artificial language)

LC Implementation Plans
Subscribers can anticipate receiving MARC records reflecting these changes in all distribution services not earlier than April 19, 2005.
