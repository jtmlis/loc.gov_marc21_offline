Appendix G - Format Changes

MARC 21 Bibliographic
April 2018

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Bibliographic Data.

MARC 21 FORMAT FOR BIBLIOGRAPHIC DATA

FORMAT CHANGE LIST

Update No. 26, April 2018
This page documents the changes contained in the twenty-sixth update to the 1999 edition of the MARC 21 Format for Bibliographic Data. Update No. 26 (April 2018) includes changes resulting from proposals that were considered by the MARC 21 community since the publication of Update No. 25 (December 2017).

Format changes in previous updates can be found on the MARC Format Overview page.
UPDATE 26 CHANGES
■ New content designators:
Codes
Code x - Not Applicable in Map 007/04 (Physical Medium)
Subfields
$3Materials specified (NR) in 377 (Associated Language)
$3Materials specified (NR) in 380 (Form of Work)
$3Materials specified (NR) in 381 (Other Distinguishing Characteristics of Work
         or Expression)
$3Materials specified (NR) in 383 (Numeric Designation of Musical Work)

■ Changes in content designator names and definitions:
Codes
Code c - Pre-production renamed Photographic pre-production
             and redescribed in Map 007/06 (Production/reproduction details)

■ Fast-track changes:
New Subfield
$4 Relationship (R) in 730 (Added Entry-Uniform Title)

Subfield
$r Total number of individuals performing alongside ensembles (NR)
         redescribed in 382 (Medium of Performance)


UPDATE 26 LOCAL PRINT GUIDE

The following sections and fields were created/changed in Update No. 26 and should be printed to keep a print copy of the format up-to-date. Changes appear in red.

  • 007 - Map

  • X30 - Uniform Titles-General Information

  • 377 - Associated Language

  • 380 - Form of Work

  • 381 - Other Distinguishing Characteristics of Work or Expression

  • 382 - Medium of Performance

  • 383 - Numeric Designation of Musical Work

  • 730 - Added Entry-Uniform Title
