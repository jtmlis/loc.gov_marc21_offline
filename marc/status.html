MARC Format Overview

This page provides summary status information relating to the MARC 21 formats and related code lists published by the Network Development and MARC Standards Office.

Official version. The Full online format is the Official standard for the format. It integrates all Updates (designated by number, month, and year) into the current base edition. The Full online format is updated twice annually. Format changes occuring since the last format update should not be used in records until 60 days after they are published in the Full online MARC format documentation.

Other versions. The online Concise version is updated at the same time as the Official (Full online) and the field lists are updated shortly thereafter. In case of any differences, the Official (Full online) is the standard.

Code lists. The Official version of the code lists are the ones available online. These are maintained continuously. Code additions or changes may be used in MARC records as indicated in the announcement published on the MARC Website.

Additional Value List Documentation

Value Lists for Codes and Controlled Vocabularies (online). Several smaller value lists, for codes and controlled vocabularies (in addition to the ones described above for countries, languages, geographic areas, organizations, and relators).

  • Official version online. Continuously updated.

Source Code List Documentation

Source Codes for Vocabularies, Rules, and Schemes (online). Source Codes contains value lists of controlled vocabularies of terms (e.g., subject terms), codes (e.g., language codes), identifiers (e.g., standard identifiers), or for specific data formulation rules (e.g., RDA) or schemes (e.g., DDC). Access provided by list type or through MARC field index.

  • Official version online. Continuously updated.
